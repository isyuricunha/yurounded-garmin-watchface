# YuRounded - Minimalist Garmin Watch Face

YuRounded is a sleek, minimalist watch face inspired by modern smartwatch design with powerful customization options.

## FEATURES (v1.0.9)
- **Large, easy-to-read numbers** with rounded design
- **13 color options** for all customizable elements
- **Battery level display** with extensive customization:
  - Percentage mode or Icon (bar) mode
  - 3 size options (Small, Medium, Large)
  - Auto color mode (alerts: red ≤20%, yellow ≤50%) or 13 custom colors
  - 3 position options (Top, Middle, Bottom)
- **Date display** (optional):
  - 4 formats: Day/Month, Month/Day, Weekday, Weekday Short
  - 4 positions: Top, Bottom, Left, Right
  - 13 color options
- **Seconds display** (optional):
  - Small font below minutes
  - 13 color options
  - Note: May increase battery consumption
- **Bluetooth indicator** (optional):
  - Shows when phone is connected
  - 4 corner positions
  - 13 color options
- **Notification indicator**:
  - Red dot when notifications are pending
  - Can be toggled on/off
- **Customizable background** (black or white)
- Clean, distraction-free interface
- Battery-efficient design
- Perfect for AMOLED displays

## CUSTOMIZATION
Choose from **13 colors** for all elements:
- Orange, Red, Blue, Light Blue, Green, Teal
- Yellow, Purple, Pink, Cyan
- White, Light Gray, Dark Gray

**Default colors:**
- Hours: Orange (#FF9966)
- Minutes: Blue Gray (#7A93AC)
- Background: Black

## DESIGN PHILOSOPHY
YuRounded focuses on what matters most: the time and essential information. No clutter, no unnecessary data - just beautiful, bold numbers with optional smart features that maintain the minimalist aesthetic. Every element is optional and customizable, allowing you to create the perfect balance between information and simplicity.

Perfect for daily wear, sports, or any activity where you need quick time checks with just the right amount of information.

## COMPATIBILITY
Supports a wide range of Garmin devices including Fenix 6/7/8 series, Forerunner 55/245/255/265/745/945/955/965, Venu series, Instinct 2/3, MARQ series, Epix, Enduro, Descent, and many more!

### Technical Details
- Minimum API Level: 3.1.0
- SDK Connect IQ 8.3.0
- Compatible with devices from 2019 onwards

## INSTALLATION
1. Visit the [Garmin Connect IQ Store](https://apps.garmin.com/apps/)
2. Search for "YuRounded"
3. Click "Install" and sync with your device
